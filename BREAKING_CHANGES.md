# Breaking Changes

This page summarizes breaking changes across all Guild versions. For details, see the release folder for your version.

- [v0.5.0 Breaking Changes](docs/releases/v0.5.0/RELEASE_NOTES.md#breaking-changes) — Installed skill directory renames (guild-memory → markdown-memory, guild-inbox → markdown-inbox, guild-issues → github-issues)
- [v0.3.0 Breaking Changes](docs/releases/v0.3.0/BREAKING_CHANGES.md) — Agent file paths renamed (thematic naming)
- [v0.2.0 Breaking Changes](docs/releases/v0.2.0/BREAKING_CHANGES.md) — Skill namespace renames (memory → guild-memory, tasks → guild-tasks, inbox → guild-inbox)
